Projects

Line Following Robot

Designed and built a line-following robot using LOGIC GATES, for autonomous navigation. Create the logic algorithm by using Karnaugh Maps, Boolean Algebra, and Truth Tables.

Hardware: , Logic Gates, Battery, L298N Motor Driver, 5V DC Motor, IR Sensors

Programming Language: None (Logic Circuit Design)

Toolchain and Methodologies: Breadboard, Multimeter, Soldering Kit, Truthtable, Karnaugh Mapping, Boolean Algebra

iLibrary: Library Management System

Designed and built a library management system that has an RFID scanner to track books and users. The system automates book check-in and check-out processes, improving efficiency and accuracy in library operations. We used RFID technology and Arduino Uno R3 for it's hardware part. Also I used database management for tracking books and users.

Hardware: SPI Interface, Arduino Uno R3, MFRC522 RFID Reader

Programming Language: C++, C# .NET, SQL

Toolchain and Methodologies: Arduino IDE, Microsoft Visual Studio 2022

Aerovest

Developed thesis study whereas we created a wearable automatic cooling vest that utilizes cooling technology to regulate body temperature of electricians and construction workers in a site. This vest features is to read physiological data such as body temperature, heart rate, and environmental data levels using various sensors. The data is then used to automatically adjust the cooling intensity of the vest to maintain optimal comfort for the wearer. It also has a desktop app whereas the head office can monitor the status of their field workers even without access to the internet.

Hardware: ESP32, DHT11, MAX30102, Battery, 5V DC Fan

Programming Language: C++, C# .NET, SQL

Toolchain and Methodologies:Arduino IDE, Microsoft Visual Studio 2022, ESPNOW Library, Diptrace

Automatic Retractable Awning

Created an solar powered automatic retractable awning system that uses rain sensors to automatically extend or retract the awning based on weather conditions. The system is designed to protect outdoor spaces.

Hardware: 24v Planetary DC Gear Motor, Rain Sensors, ESP32, DC-DC Buck Converter 24v Battery, Solar Panel, Solar Charge Controller, Fuse, Current Sensor

Programming Language: C++

Toolchain and Methodologies: Power Optimization, Arduino IDE, Multimeter, Descriptive Statistics, Soldering Kit, Schemetic Design, Welding, Woodworking Machining.

Automatic Recyclable Dispenser: Paper to Pen

Developed an automatic recyclable dispenser that converts used paper into pens. The system uses a weight sensor to measure the amount of paper inserted, and based on the weight, it dispenses a pen. This project aims to promote recycling and sustainability by encouraging the reuse of paper waste in a creative way in universities.

Hardware: Arduino Uno r3, Servo Motor, 16x2 LCD Display, Push Buttons, Weight Sensor, Load Cell.

Programming Language: C++

Toolchain and Methodologies: Arduino IDE, Multimeter, Descriptive Statistics, Soldering Kit, Schematic Design, Woodworking.
